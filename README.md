# kitchen_infographic

This is my personal kitchen cheatsheet, with all the stuff I keep looking up from time to time. 

## Build

Run:

```sh
./build.sh
```

The script installs Quarto locally into `.tools/` if needed, then renders `facts_fr.md` through `facts_fr.qmd`. The PDF is written to `dist/facts_fr.pdf`.
